## 1. Yêu cầu chức năng

Chương trình phân tích điểm học sinh:

## 2. Kết quả Unit Test

Đã xây dựng các test case sử dụng JUnit 5:

- **Test case bình thường:** Input chứa cả điểm hợp lệ và không hợp lệ.

- **Test case biên:** Danh sách rỗng (Empty List).

- **Test case ngoại lệ:** Danh sách toàn điểm sai quy định.
